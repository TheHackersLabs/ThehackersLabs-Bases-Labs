# 🧡 ThehackersLabs-Bases-Labs

Este repositorio contiene la **estructura de los archivos** para los Labs de thl. Para mantener un orden, claridad y buenas prácticas, todos los labs deben esta estructura mínima obligatoria para no perder la esencia de `thl`.

## 🏗️ Estructura 

Al pasar un labotario deberás seguir la siguiente `estructura`:

```
nombre-del-lab/
├── README.md (Writeup-Explicación-Mitigación)
├── img/
├── Dockerfile/
└── login.php/
└── index.html/
├── upload.php/ (Archivos opcionales)
```

## 🎯 Objetivo

El objetivo de los laboratorios de `The Hackers Labs (THL)` es proporcionar un entorno de aprendizaje accesible para toda la comunidad, permitiendo que cualquier persona pueda aprender ciberseguridad de forma gratuita, sin necesidad de pagar por formación, se agradece seriedad.
